## Import Steps

### 1. Download Google Ads Editor
https://ads.google.com/intl/en_au/home/tools/ads-editor/

### 2. Import in This Order
1. `campaigns.csv` — creates the 2 campaigns
2. `keywords.csv` — adds 47 keywords to 6 ad groups
3. `ads.csv` — adds 6 responsive search ads
4. `negative-keywords.csv` — blocks wasted spend

### 3. Review Before Posting
- All campaigns set to **Paused** — nothing goes live until you enable
- All Final URLs point to: `https://dianapps.com/technology-partner-australia`
- Budget: AUD $40/day (AppDev) + AUD $35/day (AI-ML) = AUD $75/day total
- Monthly budget: ~AUD $2,250

### 4. Post & Enable
- Click Post → upload to Google Ads
- Enable both campaigns when ready
